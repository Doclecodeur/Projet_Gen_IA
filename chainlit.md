# 🤖 Assistant Intelligent RAG + Agents

Bienvenue ! Je suis un assistant conversationnel capable de combiner plusieurs sources de connaissances pour répondre à vos questions.

---

## Ce que je sais faire

| Capacité | Exemple |
|---|---|
| 📄 **Documents internes** | *« Quelle est la politique de congés de l'entreprise ? »* |
| 🧮 **Calculs** | *« Calcule la racine carrée de 256 plus 18 »* |
| 🌤️ **Météo** | *« Quelle est la météo à Marseille aujourd'hui ? »* |
| 🔍 **Recherche web** | *« Quelles sont les dernières actualités en intelligence artificielle ? »* |
| ✅ **Liste de tâches** | *« Ajoute "relire le rapport" à ma liste »* |
| 💬 **Conversation** | *« Bonjour, peux-tu m'expliquer ce qu'est le RAG ? »* |

---

## Comment je décide quoi faire

Je choisis automatiquement la meilleure source de réponse :

```
Votre question
      │
      ▼
  Vos documents contiennent-ils la réponse ?
      │
  OUI → Je consulte les documents et cite mes sources
  NON → J'utilise mes outils ou réponds directement
```

---

## Astuce

Posez vos questions naturellement — je détecte moi-même si la réponse se trouve dans vos documents ou si j'ai besoin d'utiliser un outil.
